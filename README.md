# expense-management-system
The project is about the daily life expense managment like on food travel clothes etc, this toolkit is made to save time and be more aware where someone spend there money

##Steps to Install & Run the Project

Download or clone the repository to your local machine.

Ensure Python 3 is installed.

Open a terminal in the project directory.

Run the script with:
https://github.com/nayan25bai11018-dev/expense-management-system.git

Follow the command-line menu to use all features.
